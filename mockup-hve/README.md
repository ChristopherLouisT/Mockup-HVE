# 📦 How to Install and Run This App (Beginner Friendly Guide)

This guide will help you run the app on your computer step by step.\
No technical experience needed 😊

------------------------------------------------------------------------

## 🖥 Step 1 --- Install Node.js (One Time Only)

1.  Open this website:\
    👉 https://nodejs.org

2.  Click the **LTS (Recommended)** version.

3.  Download and install it like a normal application (just keep
    clicking **Next**).

4.  After installation finishes, restart your computer.

------------------------------------------------------------------------

## 📂 Step 2 --- Open the Project Folder

1.  Download or extract this project.

2.  Open the project folder.

3.  Click on the folder address bar.

4.  Type:

    cmd

5.  Press **Enter**.

A black window (Command Prompt) will appear.

------------------------------------------------------------------------

## 📦 Step 3 --- Install the App

In the black window, type:

    npm install

Then press **Enter**.

Wait until it finishes (this may take a few minutes).

------------------------------------------------------------------------

## 🚀 Step 4 --- Start the App

After installation finishes, type:

    npm run dev

Press **Enter**.

Wait until you see something like:

    Local: http://localhost:3000

------------------------------------------------------------------------

## 🌐 Step 5 --- Open the App

1.  Open your browser (Chrome recommended).

2.  Go to:

    http://localhost:3000

The app should now be running 🎉

------------------------------------------------------------------------

# ❗ If Something Doesn't Work

### If you see:

`'npm' is not recognized`

👉 It means Node.js is not installed correctly.\
Reinstall Node.js and restart your computer.

------------------------------------------------------------------------

# 🛑 To Stop the App

Click the black window and press:

    CTRL + C

Then press:

    Y

# ✅ That's It!

You're ready to use the app.\
If you close the app and want to open it again later:

1.  Open the project folder

2.  Type `cmd`

3.  Run:

    npm run dev
