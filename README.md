# 📦 How to Clone This Project Using VS Code (For Non-IT Users)

This guide will help you download (clone) the project to your computer using **VS Code Terminal** after opening the project link.

---

## ✅ Step 1: Install Required Applications

Before starting, make sure you have:

1. **Git** → Download from: https://git-scm.com/
2. **VS Code** → Download from: Microsoft website: https://code.visualstudio.com/

Install both applications completely before continuing.

---

## ✅ Step 2: Open the Project Link

1. Open the project link in your browser (usually hosted on GitHub).
2. Click the green **"Code"** button.
3. Copy the **HTTPS link**.

It should look like this:

```
https://github.com/username/project-name.git
```

---

## ✅ Step 3: Open VS Code

1. Open **VS Code**.
2. Click **File → Open Folder**.
3. Choose or create an empty folder where you want to save the project.
4. Click **Select Folder**.

---

## ✅ Step 4: Open Terminal in VS Code

1. In the top menu, click **Terminal**.
2. Click **New Terminal**.

A terminal will open at the bottom of VS Code.

---

## ✅ Step 5: Clone the Project

In the terminal, type:

```
git clone PASTE_PROJECT_LINK_HERE
```

Example:

```
git clone https://github.com/username/project-name.git
```

Then press **Enter**.

Wait until the process finishes.

---

## ✅ Step 6: Open the Cloned Project

After cloning is complete:

1. A new folder with the project name will appear.
2. Click **File → Open Folder**.
3. Select the newly created project folder.

Done 🎉 The project is now successfully cloned and opened in VS Code.

---

## ❓ If You See an Error

* Make sure Git is installed.
* Restart VS Code after installing Git.
* Make sure you copied the correct link.
* If the project is private, you must have permission to access it.

---

You’re all set! 🚀
The project is now on your computer and ready to use.
